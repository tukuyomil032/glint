# glint

glint is a Rust CLI tool for analyzing and optimizing PrismLauncher storage usage.

Main capabilities

scan disk usage
detect duplicate mods
detect unused libraries
analyze world sizes
clean launcher caches
deduplicate assets


Target users

Minecraft modpack players
server developers
large modpack testers


Supported Platforms

Windows
macOS


Non Goals

glint does not modify game data.

glint does not alter instance configuration.
